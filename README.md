# ⚽ FIFA Data Cleaning & Exploratory Analysis

## Project Overview
This project focuses on cleaning and analyzing a real-world FIFA dataset containing 100+ columns. The goal was to handle messy data and extract meaningful insights.

## Data Cleaning Steps
- Handled missing values using mean/median strategies
- Converted currency columns (Value, Wage) into numeric format
- Fixed encoding issues and removed hidden characters
- Standardized mixed units (lbs to kg in Weight column)
- Removed duplicates and cleaned column names

## Analysis Performed
- Grouped data by Club and analyzed nationality diversity
- Explored player attributes and distributions
- Created visualizations using matplotlib

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Insight
Top football clubs tend to have players from diverse nationalities, indicating global recruitment strategies.

## Conclusion
This project highlights the importance of data cleaning in real-world datasets and how proper preprocessing improves analysis quality.
